# notes/

Working notes. Not part of the doctrine, procedure, or architecture surface.

Items here may be:
- promoted to `doctrine/` if they become a locked rule
- promoted to `procedure/` if they become an execution step
- promoted to `architecture/` if they survive benchmark evidence
- deleted if they don't survive evidence

This directory is the only place in the repo where speculative thinking is welcome. Even here, anything that contradicts a locked doctrine rule should be flagged in the note itself.
