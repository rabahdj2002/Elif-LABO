01

The Cognitive Archive (Folder System)

The ELIF system organizes intelligence not as linear chats, but as Cognitive Sectors. Each sector (folder) represents a high-level domain of inquiry.

Sorting Logic

Folders are dynamically sorted by Temporal Relevance. The most recently updated Inquiry automatically floats to the top of the archive.

Unique Indexing

Every new folder is assigned a sequential Case ID (e.g., case_04) to maintain architectural integrity.

02

The 11-Step Logic Engine

When an Inquiry is initialized, it passes through an immutable reasoning pipeline that separates Operative Truth from speculative noise.

1-3

Validation & Decomposition

Checks the frame boundaries and breaks the objective into atomic logical units.

4-6

Roadmap & Hypothesis

Constructs branching paths and tracks capacity accumulation for each branch.

9-11

Governance & Audit

Applies strict constraints to the logic and logs every ambiguity for final adjudication.

03

Room Architectures

Exploration

Visualizes the branching roadmap. Each node represents a decision point in the cognitive journey.

Uncertainty

Exposes failure conditions and logical gaps. This is the "Audit Room" for potential biases.

Governance

Displays the final verdict and the rationale behind the selected trajectory.

Decision

The culmination of the Inquiry. Maps the operative path vs theoretical divergences.

04

Network Wallet & Spend

ELIF tracks every LLM token in real-time. The Network Wallet provides a granular audit of costs, broken down by Inquiry and Model usage.

Optimization Tip

Activate Offline Simulation Mode in settings to test logic flows without incurring API token costs.
