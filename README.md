# elif-lab

Experimental corrigible laboratory for **ELIF** — the corrigible discovery and validation architecture.

**Status:** Pre-prototype. One experimental data point (electroculture, Round 7). Two more cases pending before any architectural commitment.

## This repo is NOT

- The final ELIF architecture
- A civilization-scale system
- A multi-agent framework
- An OpenStudyGo extension or runtime
- A production system

## This repo IS

- An experimental laboratory
- Benchmark-driven
- Anti-architecture-inflation
- Corrigible: components may collapse, merge, downgrade, or disappear based on evidence
- Subject to its own doctrine rules (see `doctrine/02_locked_rules.md`)

## Layout

| Path | Contents |
|---|---|
| `doctrine/` | Locked governance rules, separation principles, inherited governance |
| `procedure/` | ELIF-mode procedure v0.1, benchmark template, A/B/C/D classification |
| `benchmarks/` | Benchmark tracking schemas, differentiator + component engagement |
| `cases/` | Input frames for each benchmark case (1 run, 2 pending) |
| `results/` | A/B/C outputs, comparisons, differentiator analyses (initially empty) |
| `architecture/` | Current 7-component minimum viable architecture (subject to compression) |
| `notes/` | General experimental notes |
